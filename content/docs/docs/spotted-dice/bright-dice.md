# Bright Dice

[Bright Dice](https://electricks.info/product/bright-dice/) is a stage size 8cm x 8cm x 8cm shell for Spotted Dice and the version we are selling is also compatible with [GhostMove](https://electricks.info/docs/ghostmove/). You will get one die, and 3 sets of stickers with dots, ESP symbols, and numbers. It comes with basic instructions. Created by Mike Davis, Peter Bright and Seamus Mjollnir, it is the perfect accessory if you would like to perform a routine with Spotted Dice for a larger audience.Â
