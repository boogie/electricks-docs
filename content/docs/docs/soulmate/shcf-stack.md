# SHCF Stack

This is a variation of the Si Stebbins stack, where the black suits are swapped , but the numbers remain the same.

## The Order of SHCF Stack

A♥, 4♥, 7♣, 10♦, Q♠, 3♥, 6♣, 9♦, K♠, 2♥, 5♣, 8♦, J♠,

A♥, 4♣, 7♦, 10♠, Q♥, 3♣, 6♦, 9♠, K♥, 2♣, 5♦, 8♠, J♥,

A♣, 4♦, 7♠, 10♥, Q♣, 3♦, 6♠, 9♥, K♣, 2♦, 5♠, 8♥, J♣,

A♦, 4♠, 7♥, 10♣, Q♦, 3♠, 6♥, 9♣, K♦, 2♠, 5♥, 8♣, J♦
