# Si Stebbins Stack

The  Si Stebbins Stack  is a Card Stack system originally published in the United States around 1898 by  William Coffrin  (alias Si Stebbins).

## The Order of Si Stebbins Stack

A♥,  4♥,  7♣,  10♦, Q♠, 3♥, 6♣, 9♦, K♠, 2♥, 5♣, 8♦, J♠,

A♥, 4♣,  7♦,  10♠, Q♥,  3♣, 6♦, 9♠, K♥, 2♣, 5♦, 8♠, J♥,

A♣, 4♦, 7♠,  10♥, Q♣,  3♦,  6♠, 9♥, K♣, 2♦, 5♠, 8♥, J♣,

A♦, 4♠, 7♥, 10♣, Q♦,  3♠,  6♥,  9♣, K♦, 2♠, 5♥, 8♣, J♦
